# Commands
## Build
```sh
npm run b[uild]
```
Builds `heroes.csv` from `data.yml`
## Sync
```sh
npm run s[ync]
```
syncs up the `scripts` key in `package.json` with `scripts.yml`
## Watch
```sh
npm run w[atch]
```
watches changes to `data.yml` and rebuilds `heroes.csv`
- like [Build](#build), but constantly rebuilds
