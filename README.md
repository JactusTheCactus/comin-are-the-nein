# Commands
## Build
```npm
npm run build
```
Builds `heroes.csv` from `data.yml`
## Sync
```npm
npm run sync
```
Syncs up the `scripts` in `package.json` with `scripts.yml`
## Watch
```npm
npm run watch
```
Watches changes to `data.yml` and rebuilds `heroes.csv`. (Like [`npm run build`](#build), but continuous)
