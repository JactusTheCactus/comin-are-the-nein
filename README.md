# Commands
## Build
```sh
npm run b[uild]
```
Builds `heroes.csv` from `data.yml`
## Sync
```sh
npm run s[ync]
```
Syncs up the `scripts` in `package.json` with `scripts.yml`
## Watch
```sh
npm run w[atch]
```
Watches changes to `data.yml` and rebuilds `heroes.csv`. (Like [Build](#build), but constantly rebuilds with changes)
